# AJAX with Vanilla Javascript ES6
[Reference Traversy Media](https://youtu.be/82hnvUYY6QA)

This is an example of Ajax Implementation Using pure Javascript.

## Steps
1. Create the XMLHttpRequest Object
2. Using the Object Open the Connection providing `Method and URL`.
3. Implement the onload function to execute upon request.
4. Use the Send Method to execute the ajax request.

## Ready State Values
* 0: Request Not Initialized.
* 1: Server Connection Established.
* 2: Request Received.
* 3: Processing Request.
* 4: Request Finished and Response Ready.

## HTTP Statuses
* 200: "OK"
* 403: "Forbidden"
* 404: "Not Found"
